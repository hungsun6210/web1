HTML

Markup Language (HTML) is the standard markup language for create web pages and web applications.Web browsers reseive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structures of a web pages semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered pages. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delinated by tags, written using angle brackets.
